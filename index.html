My Work üíª

A two-module subject around the math and technology behind computer graphics. This module had two coding projects that are:

  • Create a 3D software renderer with Windows GDI.
  • Create a 3D hardware render pipeline using DirectX 11.

These small demo applications were made using C++ and the Windows SDK.

A prototype of audio visualisation of audio outputted from speakers in real time within the Unity Engine. This demo uses open songs and images from the FanKit of discontinued mobile game KickFlight. Was really missing kickflight when I made this, easily my favourite mobile game back then. Wish it never closed. ‡≤•_‡≤•

Mini-games and graphics "things" done in Python and Pygame. PyGame is a python module for basic 2D game making. This was my personal introduction to OOP (though I was not that good at it back then... ).

A bunker scene demo created in a team of 6 (3 Programmers / 3 Artists) in a week for a challenge from Antimatter Games in collaboration with Derby Univsersity. Our apocalyptic bunker themed scene was made in Unity. I created the movement and object viewing functionality for this demo.

My 1st game "Robot Punch" made in Unreal Engine 5 using blueprint scripting. A short game about punching through endless robots. This game was made to focus on learning basic AI programming skills in engine.

Final year video game project. Working in a team of 9 members to develop our own video game in Unreal Engine 5 and showcased in the Derby UIni Game Expo 2023. Metta Mech is a 2 player game of mech robots fending off against an invading alien horde. My contribution to the team were to code: UI, Gameplay and Graphic effects into the game using C++ and blueprints.
