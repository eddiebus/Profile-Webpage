Some Highlights of UOD

Graphics

The graphics modules were split in two parts. Began first with learning the basics of 3D graphics and the maths behind it. Then to render simple models using windows GDI. No hardware acceleration back then, in the following 2nd Year module things moved towards DirectX programming. Graphics is easily one of favourite topics, but it took a bit of grit to get the little bits working right or for it to look the way I intended.

Networking & Security

The 2nd Year Module of Networks and Security. This assignment was on creating a system using Python and the PySocket module to create a simple network application. Students were allowed to choose their own project or resort to a proposed project. I went for 2 Player Pong. This was a challenging project that I really underestimated. The amount of times I've bottle necked or soft-locked my program on the network side was too many to count. (At least I got my own pong tho üëå)
